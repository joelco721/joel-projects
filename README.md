## Projects

### IEP Path

Live: https://ieppath.com

IEP Path is a parent-facing SaaS app that helps families organize and understand IEP/504 documents, deadlines, red flags, and next steps.

What it shows:
- Full-stack SaaS product architecture
- Authentication and protected app flows
- AI-assisted document understanding
- Multilingual product direction
- Payment/subscription implementation work
- Security-first handling of sensitive documents
- Modern responsive UI and production deployment

Tech/high-level:
- Next.js / React / TypeScript
- Supabase
- AI integration
- Vercel deployment
- Billing/subscription architecture
